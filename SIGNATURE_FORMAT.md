# Format Tanda Tangan

## Format yang Diimplementasikan

Aplikasi Invoice Generator menggunakan format tanda tangan profesional standar:

```
Hormat kami,

[RUANG KOSONG UNTUK TANDA TANGAN MANUAL]
  
Nama
_____
Jabatan
```

### Detail Format:
- **"Hormat kami"** - Opening statement
- **Ruang kosong** - 100px space untuk menulis tanda tangan manual
- **Nama** - Field editable untuk nama pemberi tanda tangan
- **Garis** - Border line tempat menulis nama
- **Jabatan** - Field editable untuk jabatan pemberi tanda tangan

### Mengapa Format Ini:
1. **Professional Standard** - Sesuai dengan standar dokumen bisnis Indonesia
2. **Editable Fields** - User dapat mengetik nama dan jabatan
3. **Manual Signature Space** - Ruang kosong untuk tanda tangan tangan
4. **Print Ready** - Optimal untuk dicetak dengan tanda tangan manual

### Penggunaan:
- **Digital**: User dapat mengetik nama dan jabatan langsung
- **Print & Sign**: Cetak dokumen, tulis tanda tangan manual di ruang kosong
- **Professional**: Format yang diakui secara bisnis dan legal
